## 動いている機能

### 麻雀ルールエンジン (`src/lib/mahjong/`)

- 牌エンコーディング (0..33), 牌種別判定 (字牌/老頭/么九/役牌)
- 山生成・シャッフル (Mulberry32 PRNG, シード再現可)
- 配牌 (13枚 + 死牌山 + ドラ表示)
- ドラ計算 (表示牌 → ドラ)
- シャンテン計算
  - 標準形 (4面子1雀頭)
  - 七対子
  - 国士無双
  - 3形式の最小値を返す
- 受け入れ計算 (ukeire)
- 和了判定、待ち牌列挙、基本フリテン判定
- 役判定
  - リーチ、ツモ、一発、海底/河底
  - 役牌、タンヤオ、平和、対々、七対子、国士
  - 一盃口、三色、一気通貫、混全帯、混一色、清一色、混老頭、三暗刻/四暗刻
  - ドラ / 裏ドラ
- 4人 no-call 対局
  - ツモ、打牌、リーチ、ツモ和了、ロン、流局
  - リーチ後のツモ切り強制
  - 簡易点数差分 (fu=30近似、満貫以上の段階処理)

### AI (`src/lib/ai/`)

- 14枚から全打牌候補を評価
- スコア = `-shanten*100000 + ukeire*200 + ukeireKinds*30 + han*80 + dora*60 - danger*weight`
- 相手リーチ時に `attack` / `balance` / `defense` を切り替える
- 守備評価
  - 現物
  - スジ
  - 壁 / ワンチャンス相当 (河・ドラ表示・自分の残り手牌を可視牌として使用)
  - 字牌の見え枚数
- 4人対戦用エージェント
  - `random`
  - `simple-shanten`
  - `attacker`
  - `world`
  - `balanced`
  - `push-fold`
  - 実験用: `risk-aware`, `value`, `solid`, `seat-aware`, `patient`

### ゲームステート (`src/lib/game/`)

- 13枚配牌 → 自動ツモ → 打牌 → 自動ツモ ... の1人練習ループ
- 河 (自分の捨て牌) の保持
- 山残り、巡目、ドラ表示の表示

### UI (`src/app/`, `src/components/`)

- 1人練習 / 4人対戦 の画面切替
- 場況パネル (局・自風・巡目・山残り・ドラ表示・ドラ・シャンテン・受け入れ)
- 手牌表示 (ツモ牌は離して強調表示)
- 牌クリックで打牌
- AI 推奨タイル枠ハイライト
- 「AI の推奨で打つ」「新しい配牌」ボタン
- 候補比較リスト (シャンテン・受け入れ・ドラ・期待打点・スコア・理由)
- 4人対戦画面
  - 各席の河、点数、リーチ表示
  - 人間席はツモ牌プレビュー付きでクリック打牌
  - AI推奨候補トップ6表示
  - 「AIを進める」「推奨で打つ」「全手牌を見る」「同じ配牌」「新しい対局」
  - 終局後の全員手牌・待ち・和了牌・放銃牌・点差表示
