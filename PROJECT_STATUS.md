## 動いている機能

### 麻雀ルールエンジン (`src/lib/mahjong/`)

- 牌エンコーディング (0..33), 牌種別判定 (字牌/老頭/么九/役牌)
- 山生成・シャッフル (Mulberry32 PRNG, シード再現可)
- 配牌 (13枚 + 死牌山 + ドラ表示)
- ドラ計算 (表示牌 → ドラ)
- シャンテン計算
  - 標準形 (4面子1雀頭)
  - 七対子
  - 国士無双
  - 3 形式の最小値を返す
- 受け入れ計算 (ukeire)
  - 13 枚 → 進める牌種・残り枚数・合計
  - 既知の捨て牌・ドラ表示牌などを除外可能
- 役の潜在評価
  - 役牌 (場風・自風・三元牌)
  - タンヤオ
  - 混一色 / 清一色
  - 平和・対々の簡易判定
  - 期待 han 数を返す

### AI (`src/lib/ai/`)

- 14 枚から全打牌候補を評価
- スコア = `-shanten*100000 + ukeire*200 + ukeireKinds*30 + han*80 + dora*60 + safety*5` − シャンテン後退ペナルティ
- 候補ランキングと、各候補の自然言語の理由を生成
- リーチ可能ならリーチを推奨
- 攻撃/守備/バランスのタグ付け (簡易)

### ゲームステート (`src/lib/game/`)

- 13 枚配牌 → 自動ツモ → 打牌 → 自動ツモ ... のループ
- 河 (自分の捨て牌) の保持
- 山残り、巡目、ドラ表示の表示

### UI (`src/app/`, `src/components/`)

- 場況パネル (局・自風・巡目・山残り・ドラ表示・ドラ・シャンテン・受け入れ)
- 手牌表示 (ツモ牌は離して強調表示)
- 牌クリックで打牌
- AI 推奨タイル枠ハイライト
- 「AI の推奨で打つ」「新しい配牌」ボタン
- 候補比較リスト (シャンテン・受け入れ・ドラ・期待打点・スコア・理由)
- 受け入れ牌一覧と残り枚数
